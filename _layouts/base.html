Use the tech you already know & love.

Simpl is a simulation platform built with Python, Django, & React.js. Simpl gives you powerful tools for gamifying your python data so that you can create fun and effective educational experiences.

100% Free & Open Source

Simpl is a completely free open source project, licensed under MIT.

Flexible & Customizable

The ability to customize the platform to your exact needs and for various simulations.

Fast & Effective

Data updates in real time; results and changes can be seen immediately.

Utilize Simpl's Core Platform Components

SIMPL-Games-API

Manages the Simpl database and provides a REST API used by the model.

SIMPL-Modelservice

Stores the data and provides relationships between data.

SIMPL-React Library

Provides UI and other front-end components
